llms.txt generator

Free browser-based tool to create an llms.txt file for your website — paste your source, generate, download. Nothing leaves your browser.

This tool builds a spec-compliant llms.txt file — a structured, machine-readable summary of your site designed for AI answer engines, voice assistants, and generative search tools (ChatGPT, Claude, Perplexity, and similar) to read and cite accurately, following the llmstxt.org standard.

How to get robots.txt, sitemap.xml, and page source

robots.txt

  1. Visit https://yoursite.com/robots.txt directly in a browser tab.
  2. Select all (Cmd/Ctrl+A), copy, paste into the box below.
  3. If the page 404s, the site has no robots.txt — that's fine, leave it blank.

sitemap.xml

  1. Check the bottom of the robots.txt you just copied — look for a line starting with Sitemap:. Visit that URL.
  2. If there's no such line, try https://yoursite.com/sitemap.xml directly.
  3. Select all, copy, paste into the sitemap box below. If the site uses a sitemap index (a sitemap of sitemaps), paste the index — the tool only reads <loc> entries, so nested sitemaps won't be expanded automatically.

Page source HTML (for every page you want documented)

  1. Open the page in Chrome/Firefox/Edge.
  2. Right-click anywhere on the page → View Page Source (or go to view-source:https://yoursite.com/page directly in the address bar).
  3. Select all (Cmd/Ctrl+A), copy the entire HTML.
  4. Click "+ Add page" below, paste the HTML, and fill in the page's actual URL in the field above it.
  5. Repeat for every important page — home, about, contact, pricing, key product/service pages. Skip pages that don't add new information.

Note: "View Page Source" (not "Inspect") gives you what the server actually sent, which is what this tool needs. Don't use browser dev tools' live DOM view — copy the raw source.
